# 系统架构图

## 系统整体架构
```
+-------------------+   +-------------------+   +-------------------+
|    用户界面层     |   |    业务逻辑层     |   |    数据存储层     |
+-------------------+   +-------------------+   +-------------------+
| - HTML/CSS/JS     |   | - Python Flask    |   | - MySQL数据库     |
| - Bootstrap框架   |-->| - 爬虫模块        |-->| - 用户表          |
| - ECharts图表     |   | - 数据分析模块    |   | - 商品表          |
| - 响应式设计      |<--| - 文本处理模块    |<--| - 评论表          |
+-------------------+   +-------------------+   +-------------------+
        ^                       ^                       ^
        |                       |                       |
        v                       v                       v
+-------------------+   +-------------------+   +-------------------+
|    安全模块       |   |    爬虫引擎       |   |   外部API接口     |
+-------------------+   +-------------------+   +-------------------+
| - 用户认证        |   | - Selenium        |   | - NLP处理接口     |
| - 会话管理        |<->| - 动态页面处理    |<->| - 数据分析接口    |
| - CSRF防护        |   | - 反爬虫策略      |   | - 可视化接口      |
+-------------------+   +-------------------+   +-------------------+
```
